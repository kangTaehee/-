## C atoi
```c
#include <stdio.h>
#include <stdlib.h>  //atoi() 함수 사용하기 위한 헤더파일
int main()
{
    char str[8] = "2021";   // 문자열 "2021"
    int num = atoi(str);    // 문자열을 숫자로 변환
    printf("%d", num * 2);   // 2021(숫자) * 2 한 값을 출력
    return 0;
}
```
atoi는 문자열을 숫자로 변환하는 함수이다.
문자열 "2021"을 숫자 2021로 변환하고 2를 곱한 값을 출력한다.
